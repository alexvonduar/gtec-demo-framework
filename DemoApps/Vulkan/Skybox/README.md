# Skybox
<img src="Example.jpg" height="135px">

<!-- #AG_DEMOAPP_HEADER_END# -->
<!-- #AG_BRIEF_BEGIN# -->
Render a simple skybox using a cubemap.
<!-- #AG_BRIEF_END# -->

For more information look at:
- [Learn OpenGL - Cubemaps](https://learnopengl.com/Advanced-OpenGL/Cubemaps)
- [Cubic Environment Mapping ](https://en.wikipedia.org/wiki/Cube_mapping)

The app can be controlled using:

Key          | Result
-------------|----------------------------------
A            | Strafe left
W            | Strafe right
S            | Move forward
D            | Move backward
Right Mouse  | Mouse look (hold it)
Middle mouse | Reset camera
